ActivityCollector

Creates an instance of an ActivityCollector

ActivityCollector($0: Object): ActivityCollector
Parameters
$0 (Object)
Name Description
$0.start Array<Number> start time obtained via process.hrtime()
$0.stackCapturer StackCapturer? (default defaultStackCapturer) which is used to decide if a stack should be captured as well as to capture and process it @see thlorenz/ah-stack-capturer The default capturer used doesn't ever capture a stack so this feature is turned off by default.
$0.requireInit boolean? (default false) when true any activities whose init hook we missed are ignored even if we see before , after and/or destroy hooks.
Returns
ActivityCollector: instance of ActivityCollector

activityCollector.dump

Dumps all so far collected activities to the console. This is useful for diagnostic purposes.

If no arguments are provided, all activities are dumped.

activityCollector.dump(opts: Object): ActivityCollector
Parameters
opts (Object) allow tweaking which activities are dumped and how
Name Description
opts.types (Array<String> | String) type(s) to dump
opts.stage String the stage to print as the title of the dump
opts.depth Number the depth with which the dumped object is dumped
Returns
ActivityCollector: activityCollector

activityCollector.dumpTypes

Dumps all types in the order they were collected including id and trigger id.

Example: FSREQWRAP:id:triggerId
